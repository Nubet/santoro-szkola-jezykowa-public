# Santoro - Szkoła Języków Obcych

Strona internetowa szkoły językowej Santoro zbudowana w `Next.js` z wykorzystaniem `React`, `TypeScript` i `Tailwind CSS`.

Odświeżona strona porządkuje najważniejsze informacje o ofercie i prowadzi użytkownika prosto do kontaktu. Projekt upraszcza komunikację szkoły, wzmacnia jej wiarygodność i lepiej pokazuje to, co naprawdę ważne z perspektywy rodzica lub kursanta.

## Co zostało zrobione

- uporządkowanie oferty językowej
- czytelny podział na typy zajęć
- prostsza prezentacja cennika
- wyraźne pokazanie lokalizacji
- mocniejsze i lepiej rozmieszczone wezwania do działania
- spójniejsza, spokojniejsza warstwa wizualna
- lepsza czytelność treści i hierarchia informacji

## Efekt

Strona jest prostsza do zrozumienia, bardziej przekonująca wizualnie i łatwiejsza w odbiorze. Lepiej prezentuje ofertę szkoły i ułatwia szybkie przejście do kontaktu.

## Uruchomienie lokalne

```bash
npm install
npm run dev
```

Aplikacja będzie dostępna pod adresem `http://localhost:3000`.

## Skrypty

- `npm run dev` - uruchamia środowisko developerskie
- `npm run build` - buduje aplikację produkcyjną
- `npm run start` - uruchamia build produkcyjny
- `npm run lint` - uruchamia ESLint
